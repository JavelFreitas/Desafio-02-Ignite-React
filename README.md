<h2 id="pre-requisitos">Pre-requisitos</h2>

* Node (v16^)
* Git
* Yarn

<h2 id="instalar-projeto">Como instalar o projeto</h2>
<h3>
Baixar projeto 
</h3>

```
git clone https://github.com/JavelFreitas/Desafio-02-Ignite-React.git
```
<h3>
Instalar as dependências 
</h3>

```
yarn
```
<h3>
Criar uma build 
</h3>

```
yarn build
```
<h3>
Rodar servidor mock 
</h3>

```
yarn server
```
<h3>
Rodar 
</h3>

```
yarn dev
```
